# VoiceOps Client Playground

## Goal

Give clients a safe way to call themselves, test the current assistant behavior, and build confidence without opening protected operator settings.

## What clients can do

- Choose a scenario preset
- Queue a self-test call
- Review recent demo sessions
- Compare captured signals with expected signals

## What clients cannot do

- Change protected compliance logic
- Change restricted tools
- Bulk-launch enrichment
- Bypass operator-owned routing rules
